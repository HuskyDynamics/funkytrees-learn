# ActivateN
---
## Syntax
`ActivateN`

## Type
Boolean

## Description
Represents the status of any one of the eight numbered Activation Groups (where `N` is 1 through 8). These are binary toggles used to enable or disable specific aircraft systems or parts.

## Valid Variables
* `Activate1`
* `Activate2`
* `Activate3`
* `Activate4`
* `Activate5`
* `Activate6`
* `Activate7`
* `Activate8`

## Range
Evaluates to `true` (1) when the group is active, and `false` (-1) when inactive.
