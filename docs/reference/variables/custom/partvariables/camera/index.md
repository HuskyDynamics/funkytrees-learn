# Camera
---

## XML ID
`Camera-1`

## Part Variables
* **IsActive**: Returns `true` (1) if this camera is the current active view.
* **LookPitch**: The current pitch angle of the camera view. (Degrees)
* **LookRoll**: The current roll angle of the camera view. (Degrees)
* **LookYaw**: The current yaw angle of the camera view. (Degrees)
* **ViewOffsetX**: The current X-axis position offset of the camera. (Meters)
* **ViewOffsetY**: The current Y-axis position offset of the camera. (Meters)
* **ViewOffsetZ**: The current Z-axis position offset of the camera. (Meters)
