# Resizable Wheel
---

## XML ID
`Wheel-Resizable-1`

## Part Variables
* **ForwardSlip**: The difference between the wheel's rotational speed and its actual forward velocity.
* **SidewaysSlip**: The lateral velocity of the wheel relative to its direction of travel.
* **Offroad**: A value indicating the current traction quality or surface type.
* **RPM**: The current rotational speed of the wheel. (Revolutions per minute)
