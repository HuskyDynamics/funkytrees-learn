# asin()
---

## Syntax
`asin(x)`

## Returns
Number

## Description
Returns the arc-sine of the input value in degrees.

## Arguments
* **x** (Number): The value to evaluate.

## Examples
* `asin(0.5)` results in `30`.
* `asin(1)` results in `90`.
