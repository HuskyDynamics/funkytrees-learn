# min()
---

## Syntax
`min(a, b)`

## Returns
Number

## Description
Compares two values and returns the smaller of the two.

## Arguments
* **a** (Number): The first value to compare.
* **b** (Number): The second value to compare.

## Examples
* `min(Altitude, 1000)` ensures the output does not exceed 1000.
