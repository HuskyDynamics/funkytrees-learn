# clamp01()
---

## Syntax
`clamp01(x)`

## Returns
Number

## Description
Restricts the input value to a range between 0 and 1. This is functionally equivalent to `clamp(x, 0, 1)`.

## Arguments
* **x** (Number): The value to be clamped.

## Examples
* `clamp01(Pitch)` ensures the output is always within the 0 to 1 range.
