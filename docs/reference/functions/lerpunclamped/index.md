# lerpunclamped()
---

## Syntax
`lerpunclamped(a, b, t)`

## Returns
Number

## Description
Performs linear interpolation between *a* and *b* by *t* without clamping the final result.

## Arguments
* **a** (Number): The starting value.
* **b** (Number): The target value.
* **t** (Number): The interpolation fraction.
