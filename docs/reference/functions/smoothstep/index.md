# smoothstep()
---

## Syntax
`smoothstep(a, b, t)`

## Returns
Number

## Description
Similar to `lerp`, but performs interpolation with smoothing at both the beginning and the end of the transition.

## Arguments
* **a** (Number): The starting value.
* **b** (Number): The target value.
* **t** (Number): The interpolation fraction.
