# deltaangle()
---

## Syntax
`deltaangle(a, b)`

## Returns
Number

## Description
Calculates the shortest angular difference between two angles in degrees.

## Arguments
* **a** (Number): The starting angle in degrees.
* **b** (Number): The target angle in degrees.

## Examples
* `deltaangle(Heading, TargetHeading)` returns the shortest turn required to face a target.
