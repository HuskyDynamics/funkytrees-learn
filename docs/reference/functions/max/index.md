# max()
---

## Syntax
`max(a, b)`

## Returns
Number

## Description
Compares two values and returns the larger of the two.

## Arguments
* **a** (Number): The first value to compare.
* **b** (Number): The second value to compare.

## Examples
* `max(Throttle, 0.5)` ensures the output is at least 0.5.
