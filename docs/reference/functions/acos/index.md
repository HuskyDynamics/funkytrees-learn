# acos()
---

## Syntax
`acos(x)`

## Returns
Number

## Description
Returns the arc-cosine of the input value in degrees.

## Arguments
* **x** (Number): The value to evaluate.

## Examples
* `acos(0.5)` results in `60`.
* `acos(0)` results in `90`.
