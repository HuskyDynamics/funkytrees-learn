# lerpangle()
---

## Syntax
`lerpangle(a, b, t)`

## Returns
Number

## Description
Similar to `lerp`, but specifically designed to interpolate between angles correctly, handling the wrap-around at 360 degrees.

## Arguments
* **a** (Number): The starting angle in degrees.
* **b** (Number): The target angle in degrees.
* **t** (Number): The interpolation fraction.
