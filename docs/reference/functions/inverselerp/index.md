# inverselerp()
---

## Syntax
`inverselerp(a, b, x)`

## Returns
Number

## Description
Calculates the linear parameter *t* (a proportion between 0 and 1) that produces the value *x* within the range [a, b].

## Arguments
* **a** (Number): The start of the range.
* **b** (Number): The end of the range.
* **x** (Number): The value to evaluate within the range.
