# atan()
---

## Syntax
`atan(x)`

## Returns
Number

## Description
Returns the arc-tangent of the input value in degrees.

## Arguments
* **x** (Number): The value to evaluate.

## Examples
* `atan(1)` results in `45`.
* `atan(0)` results in `0`.
